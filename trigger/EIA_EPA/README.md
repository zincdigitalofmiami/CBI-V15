# EIA & EPA Ingestion

Energy Information Administration (EIA) and Environmental Protection Agency (EPA) data ingestion.

## Scripts

- `Scripts/eia_procurement_ingest.ts` - EIA biofuels production data

## Target Tables

- `raw.eia_biofuels` - Biodiesel production, consumption, RFS volumes
- `raw.epa_rin_prices` - EPA RIN prices (D3, D4, D5, D6)
